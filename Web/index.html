Blindtext

Comming soon...

Block Breakers is still in development. Our goal is to complete it by the end of the year. We've already achieved a playable version with all the features of the original Tetris game, and a fairly advanced version of the 2-player version.

In the comming days we want to give it a more polished look, making good use of the Phaser framework tools. This is a trying project, and we want to make sure that everything we include in the game is working properly, so we won't add any improvements until we are sure that all the main features are running smoothly. For the next milestone, you can expect the final look of the game.
