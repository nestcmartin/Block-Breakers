Still creating...

The first version of Block Breakers has been released. Still, there's plenty of work to do. We want to keep building this awesome game into something that truly stands up with a legend such as Tetris. As future developers, we know that polish our work is essential to improve as creatives.

In the comming days we want to give Block Breaker a functional 2 player version with a couple of surprises to spice up the gameplay a little bit. Learning Phaser has been quite a journey, and we want to make sure that everything we include in the game is working properly, so we won't add any improvements until we are sure that all the main features are running smoothly.

We hope you enjoy our game. Thanks for playing.
